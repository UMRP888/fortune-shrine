# Blessing Taxonomy

Version 1.0

## Design Principle

The Shrine does not classify people.

The Shrine classifies emotional states.

A blessing should never begin from:

```text
Who are you?
```

A blessing should begin from:

```text
What are you carrying?
```

## Level 1: Primary Emotional Domains

The highest-level categories of the Blessing Corpus:

- Waiting
- Courage
- Loss
- Love
- Uncertainty
- Creation
- Return
- Renewal
- Gratitude
- Rest

## Waiting

Core feeling:

Something has not arrived.

Core imagery:

- lantern
- night
- dawn
- river
- harbor

## Courage

Core feeling:

Stepping forward despite uncertainty.

Core imagery:

- path
- mountain
- wind
- sea

## Loss

Core feeling:

Something has departed.

Core imagery:

- autumn
- rain
- river
- falling leaves
- mist

## Love

Core feeling:

Connection and affection.

Core imagery:

- moon
- lantern
- spring rain
- returning birds
- garden

## Uncertainty

Core feeling:

Not knowing.

Core imagery:

- fog
- path
- wind
- stars
- distant light

## Creation

Core feeling:

Bringing something into existence.

Core imagery:

- seed
- soil
- spring
- river source
- morning light

## Return

Core feeling:

Coming back.

Core imagery:

- harbor
- birds returning
- old trees
- evening lights
- familiar roads

## Renewal

Core feeling:

Beginning again.

Core imagery:

- spring
- rain
- seedling
- sunrise
- blooming branches

## Gratitude

Core feeling:

Appreciation for what remains.

Core imagery:

- harvest
- autumn light
- tea
- warm windows
- gentle rain

## Rest

Core feeling:

The need to stop carrying weight.

Core imagery:

- snowfall
- still water
- moonlight
- quiet forests
- winter lanterns

## Level 2: Substates

Future corpus expansion should happen primarily at Level 2.

Examples:

```text
Waiting
- Waiting For News
- Waiting For Return
- Waiting For Healing
- Waiting For Decision
- Waiting For Change
```

```text
Loss
- Separation
- Regret
- Ending
- Mourning
- Letting Go
```

```text
Uncertainty
- Crossroads
- Doubt
- Fear Of Future
- Hidden Path
- Unanswered Questions
```

## Level 3: Blessing Intensity

### Soft

Daily emotional support.

### Medium

Moments of stress, transition, or uncertainty.

### Deep

Moments of grief, loss, crisis, or profound loneliness.

## Corpus Tagging Model

Each blessing should contain:

- primary domain
- substate
- intensity
- primary imagery
- secondary imagery
- witness type
- blessing type
- echo type

Example:

```text
Domain: Waiting
Substate: Waiting For News
Intensity: Medium
Primary Imagery: Lantern
Secondary Imagery: River
Witness: Silent Witness
Blessing: Patience
Echo: Dawn
```

## Long-Term Goal

The Blessing Corpus should become a structured emotional map of human experience.

The Shrine does not categorize destiny.

The Shrine categorizes the heart.
