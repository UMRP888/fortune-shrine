Source: Blessing Corpus Alpha Parts 1-5

Processing rules:

- Original wording preserved.
- Exact duplicates removed.
- Similar but non-identical blessings kept.
- Each blessing is assigned a primary category.
- Secondary categories are listed when useful.
- Quality tags: S / A / B.

Supreme constitution:

```text
Bless the state.
Never bless the outcome.
```
